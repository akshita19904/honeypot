# Honeypot System for Attack Analysis

## Overview

Honeypot System for Attack Analysis is a low-interaction cybersecurity honeypot designed to simulate an SSH-style authentication service and capture attacker activity in a controlled environment.

The system records login attempts, securely hashes captured credentials using SHA-256, analyzes attack patterns, and visualizes security insights through an interactive threat intelligence dashboard.

---

## Features

* SSH-style honeypot authentication simulation
* Credential capture and logging
* SHA-256 password hashing
* Structured attack logging
* Brute-force attack detection
* Suspicious username detection
* Threat severity classification
* JSON-based attack analytics
* Interactive cybersecurity dashboard

---

## System Architecture

Attacker
→ Honeypot Server
→ Log Collection
→ Attack Analyzer
→ JSON Report Generation
→ Threat Intelligence Dashboard

---

## Cryptographic Concepts Used

### SHA-256 Hashing

Captured passwords are hashed using SHA-256 before being stored in logs.

Benefits:

* Prevents plaintext password storage
* Demonstrates secure credential handling
* Provides one-way protection for sensitive data

---

## Project Components

### server.py

Simulates an SSH-style login service and captures attacker credentials.

### client.py

Used to simulate attacker login attempts for testing.

### analyzer.py

Processes collected logs and generates attack intelligence reports.

### data.json

Stores structured attack analytics generated by the analyzer.

### Dashboard

Displays attack statistics, threat levels, suspicious usernames, IP analysis, and security alerts.

## Repository Setup

### Clone Repository

```bash
git clone https://github.com/as5670/honeypot-system.git
cd honeypot-system
```

### Run Honeypot

```bash
python server.py
```

### Simulate Attacks

```bash
python client.py
```

### Generate Analysis

```bash
python analyzer.py
```

### Launch Dashboard

```bash
cd frontend
python -m http.server 8000
```

Open:

```
http://localhost:8000
```
